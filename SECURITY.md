# Security Notes

This repository supports a private seminary office portal.

## Current Protection

- The deployed seed register is encrypted.
- The visible login screen does not display the password.
- The repository should remain private.
- Vercel Deployment Protection should stay enabled.

## Important

- Do not commit passwords, GitHub tokens, Vercel tokens, or student private documents.
- Revoke any token that was pasted into chat or shared outside GitHub/Vercel.
- Move to hosted authentication and a private database before using this as the permanent office system.

For office access or record corrections, contact Amazing Grace Biblical Seminary office.
